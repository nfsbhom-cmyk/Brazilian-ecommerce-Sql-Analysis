# 📊 Brazilian E-Commerce Analysis (SQL + IBM Cognos)

## 🔍 Overview

This project analyzes a real-world Brazilian e-commerce dataset to uncover insights into revenue trends, customer behavior, and product performance.

The workflow covers a complete data pipeline:
**raw data → cleaning → transformation → analysis → dashboarding in IBM Cognos**

---

## 🛠 Tools Used

* SQL (data cleaning, joins, aggregations)
* IBM Cognos Analytics (dashboard design & visualization)

---

## 📂 Dataset

* Orders
* Order Items
* Payments

---

## 🧹 Data Preparation

* Removed missing timestamps and invalid records
* Filtered negative prices and zero-value payments
* Created clean tables for structured analysis

---

## 🔗 Data Modeling

Built a unified analytical dataset by joining:

* Orders
* Order Items
* Payments

This enabled end-to-end revenue and customer analysis.

---

## 📊 Analysis Highlights

### Tables Created for Dashboarding

* Monthly Revenue
* Top Customers
* Best-Selling Products
* Multi-item Orders

### Key Metrics

* Total Revenue
* Average Order Value

---

## 📈 IBM Cognos Dashboard

### Dashboard Features

* KPI Cards: Total Revenue, Average Order Value
* Monthly Revenue Trend (Line Chart)
* Top Customers (Bar Chart)
* Best-Selling Products (Column Chart)
* Multi-item Orders Analysis (Table)

---

## 💡 Key Insights

* A small group of customers drives a large share of revenue
* Top products dominate both sales volume and profitability
* Orders with multiple items generate higher value
* Revenue trends fluctuate across months

---

## 🚀 Business Recommendations

* Focus on retaining high-value customers
* Promote top-performing products
* Encourage bundled purchases to increase order value
* Use monthly trends for demand planning
